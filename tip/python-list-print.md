# 파이썬 리스트 출력

Asterisk(*) 가 컨테이너 타입 앞에 사용되면 unpacking 된다.

```python
a = [1, 2, 2, 3, 3, 3, 4, 4, 4, 5, 5, 5, 6, 7, 8, 9]

print(*a)
print(" ".join([str(i) for i in a]))

# 1 2 2 3 3 3 4 4 4 5 5 5 6 7 8 9
# 1 2 2 3 3 3 4 4 4 5 5 5 6 7 8 9
```

언제 사용하나면, 리스트를 순회하면서 값을 출력해야할때
매번 print() 사용하지 말고, 
list를 생성하고, list를 한번에 출력한다.
